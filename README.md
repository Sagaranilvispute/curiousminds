# CuriousMinds

CuriousMinds is a student-focused platform that helps learners explore careers, study abroad opportunities, language learning, and interactive math games.

## Features

- Career Guidance
- Study Abroad Support
- Language Tutor Connections
- Interactive Math Learning Games
- Educational Resources

## Tech Stack

Frontend: Next.js
Styling: Tailwind CSS
Hosting: Vercel
Games: JavaScript

## Project Structure

src/components – reusable UI elements
src/pages – main website pages
src/games – interactive math games
public – static assets

## Deployment

The project will be deployed using Vercel with automatic deployment from GitHub.
